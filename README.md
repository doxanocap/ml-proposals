# Machine Learning & NLP Projects

Welcome to my portfolio of Machine Learning and Natural Language Processing projects. This repository contains research, experiments, and implementations focusing on Argument Mining, Text Classification, and Computer Vision.

## Projects

| Project | Description | Stack |
|---------|-------------|-------|
| **[Argument Quality Assessment](./argument-quality-assessment)** | Research project comparing SBERT, Cross-Encoders, and LLMs for argument persuasiveness ranking. | Python, PyTorch, Transformers, LLMs (GPT-4) |
| **[Negative Comments & Argument Analysis](./negative-comments-detector)** | Analysis of argument structures and sentiment, focusing on detecting persuasive vs non-persuasive discourse. | Python, Pandas, Scikit-Learn |
| **[MNIST Digit Classifier](./mnist-digit-classifier)** | Handwritten digit recognition implemented from scratch (NumPy) and with PyTorch, featuring a custom web interface. | Python, NumPy, PyTorch, JavaScript |

## Structure

- **argument-quality-assessment/**: Code and notebooks for the argument ranking research.
- **negative-comments-detector/**: Data analysis and modeling for argument/comment classification.
- **mnist-digit-classifier/**: Implementation of neural networks for digit recognition.

## Tech Stack

- **Languages**: Python, JavaScript
- **Libraries**: PyTorch, Scikit-Learn, Transformers (HuggingFace), NumPy, Pandas
- **Tools**: Jupyter Notebooks, Google Colab
